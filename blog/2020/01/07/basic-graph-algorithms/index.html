Concepts about graph

Graph Is Connected

For any two vertices, there is a path between them.

Graph H is the subgraph of Graph G

The vertices and edges of Subgraph H are subsets of the vertices and edges of Graph G.

Graph H is the spanning subgraph of Graph G

Subgraph H contains all vertices of graph G.

Forest

Graph without cycles.

Tree

Connected graph without cycles.

Spanning Tree

A spanning tree of a graph is a spanning subgraph that is a tree.

Topological Ordering

算法思路

因为图(DAG)是无环的，图中必定至少存在一个 root 顶点（in-degree is zero)。 如果删除 root 顶点以及它的出边（outgoing edges), 那么剩下的图也必定是 无环的, 即必定存在 root 顶点。因此，可以重复此删除操作，直到图变成空的。 最终按照顶点删除的顺序就可以得到一个 Topo Ordering.

伪码

def get_topological_ordering_list(dag)
  # result
  topo = []
  roots = []

  # 维护一个Map, 记录每个 vertex 的 入边数
  in_count = {}

  # 初始化 in_count
  dag.verticies.each do |vertex|
    in_count[vertex] = vertex.in_degree
    if vertex.in_degree == 0
      roots.push(vertex);
    end
  end

  ## 对应上面的 算法思路叙述
  while roots.not_empty
    u = roots.pop
    topo.push(u)

    u.outgoing_edges.each do |edge|
      v = edge.opposite(u)
      in_count[v] -= 1;
      if (in_count[v] == 0)
        roots.push(v);
      end
    end
  end

  return topo
end

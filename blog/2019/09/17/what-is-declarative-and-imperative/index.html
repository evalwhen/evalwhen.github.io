What is Declarative and Imperative

前言

之前看书经常看到某某语言是命令式语言（imperative)，但却不知其中涵义，今天在看书 Design data-intensive-application， 作者在介绍不同数据库所使用的数据模型以及其解决的问题时, 提到了不同范式的语言。

Declarative

个人理解，声明式的语言在我们使用时只关注表达 What (程序员想要的达到的目的)，不需要关注 How (即达成目的的细节)。其中典型的代表就是 SQL。 而命令式的语言则相反，使用时需要表达清楚 How(具体达成目的细节)。

举个简单的栗子，假设现在你有一个动物以及其物种的列表数据, 然后你想取得物种是鲨鱼的动物。设计成命令式的语言，可能就要写成这样： 

function getSharks() {
    var sharks = [];
    for (var i = 0; i < animals.length; i++) {
        if (animals[i].family === "Sharks") {
            sharks.push(animals[i]);
        }
    }
    return sharks;
}

但如果设计成声明式的语言，就可以简单写成这样： 

SELECT * FROM animals WHERE family = 'Sharks';
